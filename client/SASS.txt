


eliminar sass.
npm uninstall -g sass



archivos residuales borrar: 
npm cache clean --force



confirmar que sass no esta instalado.
sass --version


mirar version sass
sass --version


inicia sass para compilar. 
sass acordeon.scss acordeon.css

para ver actualizacion en time real
sass --watch acordeon.scss:acordeon.css
